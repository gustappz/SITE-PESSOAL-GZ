Meu Portfólio

Aqui estão alguns projetos simples que eu já fiz ou pensei em fazer durante meus estudos.

App Gamificado para Estudos

Um aplicativo com elementos de gamificação para tornar a experiência do usuário mais interativa e motivadora. A ideia é dar pontos para quem conclui tarefas.

  • Status: Em planejamento
  • O que usei: uma ferramenta no code e transformei em um pwa
  • O que aprendi: Lógica de pontuação e como organizar botões na tela.

Site para Minha Mãe

Um site simples criado para apresentar informações e conteúdo de forma organizada e personalizada. Fiz para ajudar ela a divulgar as coisas dela.

  • Status: Finalizado
  • O que usei: Só HTML e CSS básico
  • O que aprendi: Como escolher cores bonitas e alinhar textos para ficar legal de ler.

Site para Minha Irmã

Um site desenvolvido com visual simples e funcional, pensado para atender às necessidades dela. Tive que refazer algumas vezes até ela gostar do resultado final.

  • Status: Finalizado
  • O que usei: HTML, CSS e sistema de links
  • O que aprendi: Paciência e como escutar o que o "cliente" quer mudar no design.
